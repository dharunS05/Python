## Getting Started

To use these projects:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/dharunS05/Python.git
    ```
2. **Navigate to the folder:**
    ```bash
    cd Python
    ```
3. **Open the desired notebook with Jupyter:**
    ```bash
    jupyter notebook
    ```
